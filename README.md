# Networking-and-Packet-Analysis-Project

## Objective
Learn network commands and capture/analyze network packets on Linux using minimal tools.

## Tools Used
- WSL Linux Terminal
- netstat / ss
- nc (netcat)
- tcpdump
- ping, traceroute
- nslookup / dig

## Tasks Performed

### 1. Network Configuration
- IP addresses
- Interfaces
- Connectivity

### 2. Active Connections
- Open ports
- Listening services

### 3. Packet Capture
- Capture 10 packets
- Analyze ICMP, TCP, and DNS traffic

### 4. DNS and Routing
- Traceroute to google.com
- DNS lookups
